= DATE_TRUNC

Truncate a date or datetime value to the start of a given unit of duration. 

== Syntax
----
DATE_TRUNC( <unit>, <datetime> )
----

== Usage

Support unit of time:

- MILLENNIUM
- CENTURY
- DECADE
- YEAR
- MONTH 
- QUARTER 
- WEEK        
- DAY
- HOUR
- MINUTE
- SECOND
- MILLISECOND
- MICROSECOND
- NANOSECOND

* Returns NULL if one of the arguments is NULL.
