= IS_DATE

Check if a value is a valid date or timestamp.

== Syntax
----
IS_DATE( value, format )
----

== Usage

Return FALSE if a value is NULL. 

== Demo
----
IS_DATE( INVOICE_DATE, 'YYYY-MM-DD' )
----
